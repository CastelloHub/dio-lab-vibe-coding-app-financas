# App de Finanças Pessoais com IA - Vibe Coding

## 1. Conceito do Projeto
O **FinanzaIA** é um assistente pessoal de finanças que utiliza IA para automatizar o controle de gastos e planejamento de economia através de conversas naturais. O objetivo é eliminar a entrada manual cansativa e substituir por um diálogo intuitivo.

## 2. Prompt Final (PRD) usado com a IA
"Atue como um Especialista em Produto (PM). Crie um PRD para um app de finanças pessoais focado em 'Vibe Coding'. Requisitos:
1. Interface de Chat: O usuário deve registrar gastos falando ou digitando (ex: 'Gastei 50 reais no almoço').
2. Categorização Automática: A IA deve classificar a despesa em tempo real.
3. Agente de Economia: Um agente que analisa o saldo e sugere onde economizar 10% no mês.
4. Dashboard Visual: Gráficos simples gerados sob demanda via chat."

## 3. Resumo da Solução
- **Stack:** React + Node.js (Conceitual via Lovable).
- **IA:** Agentes inteligentes para análise preditiva e categorização.
- **Interação:** Foco total em UX conversacional.

## 4. Reflexão sobre o Processo
- **O que funcionou bem:** A tradução de intenções naturais para requisitos técnicos foi extremamente rápida usando o conceito de Vibe Coding.
- **O que não funcionou como esperado:** A definição de limites de gastos precisou de prompts mais específicos para não ser vaga.
- **O que aprendeu:** Programar conversando com a IA exige clareza na intenção, transformando o desenvolvedor em um orquestrador de soluções.
